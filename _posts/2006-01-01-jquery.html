John Resig's jQuery makes web sites and applications interactive and elevates Javascript as the 'third standard technology' beside CSS and HTML.

The nature of the web lets individuals, more than organizations and governing bodies to shape its future. Resig's invention adds a new dimension to the web: programmability to complement structure and style.

Javascript is the native programming language of the web since the first true web browser, Netscape, and hit mainstream with jQuery which makes easy to add subtle navigation to menus, to hide and reveal content in a smooth manner, to refresh on-the-fly data on websites like the content of your shopping cart.

Beside lifting the user experience on the web to a highly enjoyable state, jQuery made web developers worldwide to forget general purpose programming languages when programming for web and use Javascript. A set of new Javascript frameworks are spinning up regularly to collect, prepare, structure and deliver web data. Javascript is what glues together information when traveling from servers to our eyes.
